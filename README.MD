#H3 PUSH PARA O GITHUB

* Ciramos o arquivo via Git Bash: vim README.md
* Salvamos o arquivo: :wq
* Vemos o Status do arquivo local: git status
* Adicionamos o arquivo no git: git add
* Commit no arquivo: git commit -m "Subindo README"
* vamos o push desse arquivo: git push origin master.

#h4 Obs: Não foi criado nenhum Brash para essa primeira entrega.
