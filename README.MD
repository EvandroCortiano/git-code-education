## PUSH PARA O GITHUB

1. Ciramos o arquivo via Git Bash: vim README.md
2. Salvamos o arquivo: :wq
3. Vemos o Status do arquivo local: git status
4. Adicionamos o arquivo no git: git add
5. Commit no arquivo: git commit -m "Subindo README"
6. vamos o push desse arquivo: git push origin master.

### Obs: Não foi criado nenhum Brash para essa primeira entrega.
